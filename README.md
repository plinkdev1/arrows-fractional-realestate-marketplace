> **An AI-powered fractional real-estate marketplace.** Buy and trade shares of vetted properties, with AI-driven property insights and a portfolio dashboard that tracks fractional holdings.

## Features

| Module | Description |
|---|---|
| **Fractional ownership** | Acquire and trade fractional shares of listed properties. |
| **AI property insights** | AI-assisted valuation and analysis surfaced on each listing. |
| **Marketplace** | Browsable, filterable catalog of investable properties. |
| **Property detail** | Per-property breakdown: financials, share availability, insights. |
| **Investor portfolio** | Dashboard tracking holdings and performance across properties. |

## Engineering highlights

- **Marketplace + portfolio architecture** — listing discovery on one side, holdings tracking on the other, sharing a property/share data model.
- **AI insights layer** — property analysis integrated into the listing and detail views.
